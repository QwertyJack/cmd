# Colored Markdown

### Requirements
* PHP
* composer
    - To install [cebe/markdown](https://github.com/cebe/markdown#installation-).

### Usage

* Install cebe/markdown
```
composer require cebe/markdown "~1.0.1"
```

* Install mathjax
```
tar zxf 2.7.0.tar.gz
```

* BOOM !!!
```
cat demo.md | php parser
```
